# PatternT

Symbolic calculator  
Reduces expressions using pattern-based rules  

## Examples

rules:  
```
a + a -> a
a + b -> b
```
produce:
```
x + (x + x) -> x
(y + x) + y -> y
```

For more examples, see match tests: https://github.com/Donaim/PatternT-tests/blob/master/examples/test1  
But they also include compiler specific cases
